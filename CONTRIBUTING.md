# Contributing to ElectroManager

Thank you for your interest in contributing to ElectroManager! We welcome contributions from everyone.

## 🚀 How to Contribute

1.  **Fork the Repository**: Click the 'Fork' button at the top right of this page to create your own copy of the project.
2.  **Clone your Fork**:
    ```bash
    git clone https://github.com/YOUR_USERNAME/ElectroManager.git
    ```
3.  **Create a Branch**:
    ```bash
    git checkout -b feature/your-feature-name
    ```
4.  **Make your Changes**: Implement your feature or fix.
5.  **Commit your Changes**:
    ```bash
    git commit -m "Add some feature"
    ```
6.  **Push to your Fork**:
    ```bash
    git push origin feature/your-feature-name
    ```
7.  **Open a Pull Request**: Go to the original repository and click 'New Pull Request'.

## 🛠️ Development Setup

Follow the instructions in the [README.md](README.md) to set up the project locally.

## 📜 Code of Conduct

- Be respectful and professional.
- Use clear and descriptive commit messages.
- Ensure your code follows the existing style of the project.

## 🤝 Need Help?

If you have questions or need guidance, feel free to open an Issue or contact the maintainers.

Happy coding! ❤️
